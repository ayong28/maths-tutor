**Tips for practice:**
- Like terms have the same variables with the same powers
- Combine only the numbers (coefficients) in front of like terms
- Leave the variable part unchanged
- Organize your work: group like terms together before adding
- Terms with different variables cannot be combined (e.g., 3a + 2b stays as 3a + 2b)
- Watch for different powers: x and x² are NOT like terms
- Example: 3a + 2b + 4a = (3a + 4a) + 2b = 7a + 2b
