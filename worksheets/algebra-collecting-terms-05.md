**Tips for practice:**
- Combine only like terms - they must have the same variables and powers
- Add the coefficients (numbers) in front of like terms
- Keep the variables and their powers unchanged
- Organize by grouping like terms: (3a + 6a) + 4b = 9a + 4b
- Different variables cannot be combined (a and b stay separate)
- Powers matter: x and x² are different, so keep them separate
- Example: 7x + 5y + 2x = (7x + 2x) + 5y = 9x + 5y
