**Tips for practice:**
- When multiplying algebraic terms, multiply the coefficients (numbers) and combine the variables
- When the same variable is multiplied, add the powers (e.g., x × x = x¹⁺¹ = x²)
- Remember: x = x¹, so x × x² = x¹⁺² = x³
- For powers of expressions like (2a)², multiply the coefficient by itself: 2² = 4, and square the variable: a² = a²
- For (2a)³, calculate 2³ = 8 and a³ = a³, giving 8a³
