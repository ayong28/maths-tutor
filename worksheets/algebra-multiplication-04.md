**Tips for practice:**
- Multiply all the coefficients (numbers) together
- For variables: when multiplying the same variable, add the exponents
- Remember: x × x = x², x × x² = x³, and so on
- For expressions in parentheses like (6a)²: calculate 6² = 36 for the coefficient, a² = a² for the variable
- Example: 4x × 6x² = (4 × 6) × (x × x²) = 24 × x³ = 24x³
- Power rule: when multiplying same bases, add the exponents together
